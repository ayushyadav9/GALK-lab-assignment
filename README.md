### **Installation**

1. Clone the client repository 
   ```sh
   git clone https://github.com/ayushyadav9/GALK-lab-assignment
   ```

2. Install all npm packages from package.json
   ```sh
   npm install
   ```

3. Run the react app
    ```sh
    npm start
    ```
4. App will be runing at http://localhost:3000
